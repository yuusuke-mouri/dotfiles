# 「できない」を「できる」に変えてくれた生成AI ─ 実装型IT/DXコンサルタントの活用スタイル

## はじめに

今日は日曜日なので、日記的なコンテンツを。

最近よく「AIって使ってますか？」と聞かれます。  
実装型コンサルタントとして、私がどうAIを使っているかを書いてみます。

* * *

## 基本方針：どれか一つに$200課金

会話型AIは複数ありますが、私は「どれか一つに$200課金する」という運用をしています。

現在は **Claude MAX** （$200/月）を契約中。

なぜ一つに絞るかというと、もちろん費用面もありますが、複数に分散するとどれも中途半端になるからです。  
一つに集中して使い倒す方が、そのツールの得意・不得意が分かります。

## Claude を選ぶ理由

基本的に Claude の方が私に合っています。  
文章の場合は文体、コーディングの場合はスピード感。  
コーディングは雑な傾向があるので Gemini や Codex をレビュアーとして使ってます。

Codex CLI（ChatGPTのコーディング支援）に興味があって ChatGPT Pro（$200/月）を契約したこともありましたが、ちょっと安全に寄りすぎていると感じて、Claude Code に戻りました。

ちなみに Gemini CLI (Gemini 3 pro) はソースコード全体を読み込むような作業をさせたせいか、思ったより課金が高くて封印中です。。

他のモデルに興味がある機能が出れば、また試します。  
AI は進化が早いので、固定せず柔軟に。

## Claude Code で何をしているか

主な用途：

  * **コーディング** : 小さなツールや検証用コード、Webアプリの開発

  * **インフラ操作** : Docker、gcloud コマンドなど

  * **文章の壁打ち** : このアドベントカレンダーもClaude Codeで書いています

特にコーディングとインフラ周りは、生産性が「測れないくらい」上がっています。

なぜ「測れない」かというと、生成AIがなければ「結果ゼロ」だったアイデアが、形になるからです。

「やりたいけど時間がない」  
「調べるのが面倒」  
「ちょっと試すには重い」

こういうアイデアが、AIとの対話で実現できる。  
これは時間短縮というより、「できなかったことができるようになった」という質的な変化です。

## AIと共同で文章を書くときに気をつけていること

このアドベントカレンダーはClaude Codeと一緒に書いています。

気をつけているのは、**自分の視点を入れること** 。

AIに丸投げすると、どこかで読んだような文章になります。  
「私はこう思う」「私の経験では」という部分は、自分で書いてます。

AIは構成を整えたり、抜け漏れを指摘してくれる優秀なアシスタント。  
でも、最終的な判断と視点は自分のものです。

## 投資対効果

円安もあり、月$200は安くない金額です。  
でも、投資対効果は非常に高いと感じています。

特に：

  * コーディングや IaaS 操作 (gcloud コマンドなどでできること)

  * Docker 環境の構築

  * 各種エラーの原因調査

  * 文章作成、レビュー

  * 画像生成

特にコマンドラインの操作に慣れてる私は Claude CLI をよく使っていて、Pro ($20) だとすぐに上限に達してしまうため Max ($200) にせざるを得ない状況です。

これらは以前なら何時間もかかっていた作業、もしくは諦めていた作業が、本当に数分で終わります。  
この note も生成 AI がなければ「たぶん無理」と手を付けてませんでした。

0 を 1 に変えてくれる生成 AI はこれからももっと上手に活用できるよういろんなことを試したいと思ってます。

## その他の AI 関連ツール

全部挙げられませんが、サブとして Notebook LM を活用しています。

また試したけど定着しなかったものや、メインの生成AIが進化して使わなくなったものが多くあります。Perplexity が代表です。Deep Research は Gemini を使っています。

あとは Genspark, Devin など使ってみましたが、当時はいまいちの体験だったのですぐ解約しました。

挙げるときりが無いのでここまでにしますが、興味のあるツールはできるだけ触るようにしています。

* * *

## まとめ

私のAI活用スタイル：

  1. **$200 は一つだけ** : 各社の状況に応じて $200 課金先を決める (今は Claude Code があるので Claude)

  2. **柔軟に乗り換え** : 良いものが出れば試す。月$20程度なら出し惜しみしないで使ってみる

  3. **自分の視点を入れる** : AIに丸投げするといわゆる「こたつ記事」感が拭えない

  4. **投資対効果を意識** : 時間短縮だけでなく「できなかったことができる」ようになったという価値は大きい

まだ競争が落ち着かない様子の生成 AI 界隈ですが、トレンドを追いすぎると本業が出来なくなるくらいなのでほどほどにしています。

今のところ**文章やコーディングは Claude** で、その他は **Google Workspace + Gemini** というスタイルで落ち着いてます。

私にとっては主にコーディング領域で「できない」を「できた」に変えてくれた生成 AI ですが、もっといろんな領域で「できた」に変えてくれるポテンシャルを持っているはずなので、今後も楽しみです。

明日から残り4日間はアドベントカレンダーのまとめに入りたいと思います。
