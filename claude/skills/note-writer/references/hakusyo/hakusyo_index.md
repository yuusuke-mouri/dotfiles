# 中小企業白書 2025年版 目次

## 概要（2025年版のキーメッセージ）

円安・物価高の継続、「金利のある世界」の到来による生産・投資コスト増、構造的な人手不足など、中小企業・小規模事業者が直面する状況は依然として厳しい。

**第1部のポイント**:
- コストカット戦略は限界を迎えている
- 積極的な設備投資・デジタル化と、適切な価格設定・価格転嫁の推進が必要
- 付加価値や労働生産性を高める経営への転換が求められる

**第2部のポイント**:
- 経営者の「経営力」の向上が重要
- 売上高規模ごとに異なる「成長の壁」の打破が必要
- M&A、イノベーション、海外展開が企業規模拡大に有効
