## コンテンツポリシーチェックリスト

記事を書く前に確認すること:

- [ ] **誰かを否定していないか？**
  - 競合コンサルタント、ベンダー、ツールを攻撃していないか
  - 「〇〇は失敗する」ではなく「仕組み上、〇〇が起きやすい」

- [ ] **想定読者に刺さるか？**
  - 経営層: 投資対効果、他社事例、経営者の決断
  - 経営企画: データ・図表、社内説得のロジック
  - 士業: 顧問先への提案ネタ、連携の可能性
  - 金融機関: 取引先支援の具体策
  - 自治体: 地域経済への貢献

- [ ] **「私なら」より「技術があれば」になっているか？**
  - 自分の優位性アピールではなく、技術の価値を一般論として伝える
  - 「一緒に」「補完関係」を強調

- [ ] **出典を明記しているか？**
  - 白書の章・節・図表番号を記載
  - 「中小企業白書2025によると」と明示
