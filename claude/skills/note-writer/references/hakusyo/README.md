# 中小企業白書データ（hakusyo）

note記事作成時に活用する中小企業白書・小規模企業白書（2025年版）のデータ。

## いつ使うか

以下のテーマで記事を書くとき、このディレクトリのファイルを参照する:

- DX・デジタル化
- 価格転嫁・値上げ
- 賃上げ・人件費
- 人手不足・採用
- 事業承継・後継者
- 経営戦略・経営力
- GX・脱炭素
- M&A・スケールアップ

## ファイル一覧

| ファイル | 内容 | 用途 |
|----------|------|------|
| `hakusyo_index.md` | 白書の目次・構造 | 該当する章・節の特定 |
| `case_studies.md` | 事例一覧（18社） | 記事の具体例として引用 |
| `note_topics.md` | 記事ネタ集 | テーマ選定・構成案 |

## 使い方

### 1. テーマに関連する章を特定

`hakusyo_index.md` のキーワード検索ガイドで該当章を確認。

### 2. 事例を選ぶ

`case_studies.md` から★★★の注目事例を優先的に検討:
- 倉岡紙工（身の丈DX）
- 広島メタルワーク（他社連携DX）
- 岡三屋（適正価格設定）
- 側島製罐（MVV策定）
- 内池建設（経営者リスキリング）
- マツモトプレシジョン（事業承継×DX×GX）

### 3. 記事ネタを確認

`note_topics.md` でターゲット別の切り口とPainを確認。

### 4. 詳細データが必要な場合（RAG検索）

**Dify API で白書を検索**:

```bash
curl -s -X POST 'https://dify.ravi.fukuoka.jp/v1/chat-messages' \
  -H 'Authorization: Bearer app-2ZWUFIhY2QMuc6OKom6VQkev' \
  -H 'Content-Type: application/json' \
  -d '{
    "inputs": {},
    "query": "【検索したい内容】",
    "response_mode": "blocking",
    "conversation_id": "",
    "user": "note-writer"
  }'
```

**レスポンス例**:
- 白書からの回答
- 出典（章・節・図表番号）
- 参照したドキュメント情報

### 5. PDFを直接参照（バックアップ）

```
~/Data/Obsidian/yuusuke.ap/02_Projects/Active/note.com2025AdventCalendar/中小企業白書2025/
├── 中小企業白書/
└── 小規模企業白書/
```

## 出典の書き方

記事内で白書データを引用する際は必ず出典を明記:

```
中小企業白書2025によると、〇〇は△△%となっている（第1部1章5節）。
```

## 注意事項

- データは2024年度までの統計に基づく
- 「平均」は中小企業の多様性を代表しない可能性あり
- 白書にないデータは「白書には該当データがありません」と明記
