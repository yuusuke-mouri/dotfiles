## 追加ルール

### 新しい「約束」を記録
記事を書く際、「また今度」「いずれ」などの言及があれば、このファイルに追加すること。

### 消化したテーマ
テーマを消化（記事化）したら、以下の形式で記録:

```markdown
### [テーマ名] ✅ 消化済み
- **消化記事**: [タイトル](URL)
- **公開日**: YYYY-MM-DD
```

## 優先度の目安

| 優先度 | テーマ | 理由 |
|--------|--------|------|
| 高 | 経営理念「在るものを尊び」 | 価値観の深掘り、ブランディング |
| 高 | 生成AIに仕事を奪われたら | 時事性、関心が高い |
| 中 | Excel WBSはやばい | 具体的エピソード、共感を得やすい |
| 中 | 管理は誰でもできると思っていた | キャリア考察 |
| 低 | 会社を辞めてから発信を始めるまで | 個人的な話、ターゲット外の可能性 |
