## 接続詞の使い分け

### 現状の課題
「でも」の多用はカジュアルすぎる印象を与える。権威性と親しみやすさのバランスを取る。

### 使い分けガイド

| 場面 | カジュアル（控えめに） | バランス型（推奨） |
|------|----------------------|-------------------|
| 逆接 | でも | しかし、一方で、ただ |
| 因果 | だから | そのため、したがって |
| 追加 | あと、それと | また、加えて |
| 例示 | 例えば | たとえば、具体的には |
| 転換 | で、 | さて、ところで |

### ルール
- **「でも」は1記事3回まで**。残りは「しかし」「一方で」を使う
- 同じ接続詞を2段落連続で使わない
- 文頭の接続詞は段落の1/3程度に抑える（多すぎると幼稚な印象）

## 権威性・信頼性を出す表現

### 経験への言及パターン
- 「27年の現場経験から言えるのは〜」
- 「私がこれまで見てきた中では〜」
- 「現場で何度も遭遇するパターンとして〜」

### データ・出典の引用
- 「〇〇調査（2026年）によると〜」
- 「経済産業省のレポートでは〜」
- 具体的な数字を入れる（「多くの」→「約7割の」）

### 断定と推量のバランス
| 場面 | 表現 |
|------|------|
| 核心的な主張 | 断定（「〜です」「〜である」） |
| 応用・一般化 | 推量（「〜かもしれません」「〜の場合が多い」） |
| 自分の経験 | 限定（「私の経験では」「少なくとも〜では」） |

### 謙虚さと自信のバランス
- NG: 「〜と思います」の多用（弱い印象）
- OK: 「〜と考えています」「〜というのが私の見解です」
- OK: 「私の経験では〜です」（限定しつつ断定）

## 避けるべきパターン

### 「頭悪そう」に見える表現
- 同じ接続詞の連続使用（でも→でも→でも）
- 「〜と思います」「〜な気がします」の多用
- 主語の省略しすぎ（論理が追いにくい）
- 「すごく」「めっちゃ」「やばい」（本文では避ける）

### 権威性を損なう表現
- 過度な謙遜（「私なんかが言うのもなんですが」）
- 責任回避（「間違っているかもしれませんが」）
- 曖昧な数量（「たくさん」「けっこう」→ 具体的な数字に）
